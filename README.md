### **AIM**

To study and implement basic Natural Language Processing (NLP) techniques for processing and analyzing textual data, including tokenization, stop-word removal, stemming/lemmatization, and feature extraction.

---

### **THEORY**

Natural Language Processing (NLP) is a branch of artificial intelligence that enables computers to understand, interpret, and generate human language. It combines computational linguistics with machine learning and deep learning techniques.

Key NLP concepts include:

* **Tokenization**: Splitting text into smaller units like words or sentences.
* **Stop-word Removal**: Eliminating common words (like *is, the, and*) that do not add meaningful information.
* **Stemming**: Reducing words to their root form (e.g., *running → run*).
* **Lemmatization**: Converting words to their base form using vocabulary and grammar (e.g., *better → good*).
* **Vectorization**: Converting text into numerical form using methods like Bag of Words or TF-IDF.

These preprocessing steps help improve the performance of NLP models in tasks such as sentiment analysis, text classification, and information retrieval.

### **ALGORITHM**

1. Import necessary NLP libraries (e.g., NLTK, spaCy).
2. Input the text data.

5. Remove stop words from the text.
6. Apply stemming or lemmatization.
7. Convert processed text into numerical format (TF-IDF or Count Vectorizer).
8. Analyze or display the processed output.
